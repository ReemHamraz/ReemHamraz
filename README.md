## About

I'm a CS undergrad at Integral University (Data Science & AI) who works at the edge of AI and real-world impact — from clinical research to open-source astronomy tooling.

Currently: **Google Summer of Code 2026 @ OpenAstronomy**, building a robust C/Cython testing framework for Astropy.

I've co-authored a **published research paper** on post-liver-transplant weight trajectories using generative AI, built production-grade ML systems, and won a national urban innovation hackathon. I care about healthcare AI, infrastructure that doesn't break, and shipping things that actually work.

## What I've Been Up To

**GSoC @ OpenAstronomy** *(2026)*
Architected an isolated `pytest` framework for Astropy's C/C++/Cython layer, refactored C-level circular dependencies to unblock the Meson build migration, and authored documentation mapping the C-API testing surface across six core modules.

**Research @ Virginia Commonwealth University** *(2024 – present)*
Analysed 562 liver transplant cases in R Studio, built 25+ statistical models, and co-authored a clinical paper on AI-driven metabolic risk assessment post-transplant.

**AI Intern @ Signature IT** *(2025)*
Built a multi-model classification system (SVM / KNN / Decision Tree) hitting 92% accuracy, automated an ML pipeline that cut prediction error by 18%, and shipped a real-time Python analytics dashboard.
